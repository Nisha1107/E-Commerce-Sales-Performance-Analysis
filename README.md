# E-Commerce-Sales-Performance-Analysis
End-to-end E-commerce Sales Performance Analysis using Python and Power BI.

Project Overview:

This project focuses on analyzing e-commerce sales performance using a realistic dataset to uncover key business insights related to sales trends, product performance, customer preferences, and seasonality. The project follows a complete end-to-end data analytics workflow using Python for data analysis and Power BI for interactive dashboards.

Tools & Technologies:

* Python: Pandas, Matplotlib, Seaborn
* Jupyter Notebook
* Power BI Desktop
* Excel / CSV

Project Structure:

├── ecommerce_sales_cleaned.csv
